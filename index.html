mScroll plug-in demo. / mModalプラグインデモ

This is demo page of mScroll plug-in. Click the following links.
If you click the <a> elements have mScroll method,
window smoothly scroll to the element that was specified by href attribute of the <a> element.

mScrollプラグインのデモページです。
mScrollメソッドを指定している<a>要素をクリックすると、
href属性で指定された要素へスムーズにスクロールします。

  • $('a').mScroll()
  • $('a').mScroll(200)
  • $('a').mScroll(null, 'easeInExpo')
  • $('a').mScroll(null, null, -100)

Please use this plug-in via GitHub. / GitHubからご利用ください。
